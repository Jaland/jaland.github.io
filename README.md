# Personal Blog

See it live at https://jaland.github.io

## Build Locally

### Prerequisites

```sh
sudo dnf install ruby ruby-devel openssl-devel redhat-rpm-config gcc-c++ @development-tools
```

### Build Locally

```sh
bundle install
bundle update github-pages
bundle exec jekyll serve
```
