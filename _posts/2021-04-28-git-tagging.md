# Create Git Tag


## Create tag

```bash
git tag -a v1.0 -m "Eample of creating a git tag for version v1.0 with a comment"
```

## Push a specific tag

```bash
git push origin v1.0
```

## Push all your tags

```bash
git push origin --tags
```
